title: Como usar Javascript?
description: A reference page in my new Starlight docs site.
---

- En un navegador → Usando la consola de desarrollo o insertando código en <script> dentro de HTML.
- En un editor de código → Como VS Code, donde puedes escribir y ejecutar scripts en un navegador.
- En un entorno de servidor → Con Node.js, que permite ejecutar JavaScript fuera del navegador y crear servidores o aplicaciones backend.
- En un sandbox online → Como JSFiddle, CodePen o Replit, donde puedes probar código sin instalar nada.
