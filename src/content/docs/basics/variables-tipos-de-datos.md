## Concepto

Sirven para guardar informacion.

```javascript
let nombre = "Edwin";
const edad = 30;
let activo = true;
```

## Ejercicio

Crea variables:

- tu nombre
- tu edad
- si eres estudiante (true/false)

Imprime en consola:

```javascript
console.log(nombre, edad, activo);
```
