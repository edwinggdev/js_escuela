## Concepto

Permiten hacer calculos y comparaciones.

```javascript
let a = 10;
let b = 5;

console.log(a + b); // suma
console.log(a > b); // comparacion
```

## Ejercicio

Calcula:

- suma
- resta
- multiplicacion

Compara si un numero es mayor que otro.
