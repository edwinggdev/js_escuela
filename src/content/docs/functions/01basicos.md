Variables (let, const, var)
- let: Declara una variable que puede cambiar su valor.
- const: Declara una variable que no puede cambiar su valor.
- var: (No recomendado) Funciona similar a let, pero con un alcance menos seguro.

```javascript
let nombre = "Juan"; // Puede cambiar
const edad = 30; // No puede cambiar
```
