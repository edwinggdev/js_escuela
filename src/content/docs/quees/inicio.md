---
title: Que es Javascript?
description: Un breve resumen.
---

JavaScript es un lenguaje de programación usado principalmente para desarrollar sitios web interactivos. Permite manipular el contenido de una página, gestionar eventos y comunicarse con servidores. Es uno de los pilares del desarrollo web junto con HTML y CSS.
